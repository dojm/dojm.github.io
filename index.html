Commenting & Voting App with Facebook Login

Visit Demo

This is the final project for my class in Advanced Web Applications with PHP.

Logged in users can leave a comment and vote on other people's comments. It is built using the Laravel framework. Users can log in via their Facebook account, which we manually built a log in flow for (instead of using an SDK). Data is stored in a MySQL database.

Please Note: Due to COVID-19, the Facebook Login feature will not work, as Facebook has halted verification of new developer accounts during this time (this means it will not work live). Please use "Log In as Guest" for the time being to view the project.

UX/UI Class Final Project

View Demo

This was our final project for the class UX/UI Fundamentals. The deliverable was a presentation and a document (40+ page PDF) of our design specifications including research, workflow, iterations of prototypes, and final mock up for a fictitious application.

I delegated tasks to each member of our group according to each member's skillset to deliver the best possible product. My main tasks for the final assignment included creating logic flows and compiling data from user tests and from every iteration of our application.

Adobe XD was used for prototyping. We went through several iterations of our application, with the user being at the center of our design to provide the best experience based on their needs. The demo shows a sample of our final project.

Food & Beer Pairer using Punk API

View Code View Demo

This is a project I did for practice working with APIs and ReactJS. This tool helps the user find a recommended beer to pair with their meal. It uses the Punk API which contains a catalogue of Brewdog beer (a brand of craft beer from the UK).

The user enters in a food item, then recommended beers are fetched according to the query. I made this with plain Javascript, then a second version using ReactJS. The demo uses ReactJS.
